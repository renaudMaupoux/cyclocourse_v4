│ ⚙️ FONCTIONNALITÉS PRINCIPALES                                               │
└───────────────────────────────────────────────────────────────────────────────┘

🎤 Reconnaissance Vocale
   → Upload fichiers audio (WAV, MP3, FLAC)
   → Détection automatique chiffres 1-1000
   → Langue: Français (fr-FR)
   → Confiance de reconnaissance affichée

🎨 Gellules Intelligentes
   → Affichage sous forme de capsules colorées
   → Code couleur selon la confiance:
      🔵 Bleu   = Haute confiance (>90%)
      🟠 Orange = Moyenne (70-90%)
      🔴 Rouge  = Faible (<70%)
   → Édition inline (clic sur la gellule)
   → Indicateur visuel si modifié manuellement

⚡ Temps Réel (WebSocket)
   → Synchronisation automatique entre clients
   → Notifications instantanées
   → Reconnexion automatique

📊 Statistiques
   → Total de chiffres détectés
   → Répartition par niveau de confiance
   → Nombre de modifications manuelles

Démarrage automatique:
   .\start-dev.ps1

Démarrage manuel:
   Terminal 1: cd backend && npm run dev
   Terminal 2: cd frontend && npm run dev

Installation dépendances:
   cd backend && npm install
   cd frontend && npm install

Réinitialiser la base:
   Remove-Item backend\database.sqlite
   (puis redémarrer le backend)

┌───────────────────────────────────────────────────────────────────────────────┐
│ 🎓 PREMIERS PAS                                                               │
└───────────────────────────────────────────────────────────────────────────────┘

1. Lancez l'application avec .\start-dev.ps1

2. Testez avec un fichier audio:
   - Cliquez sur "📁 Choisir un fichier audio"
   - Sélectionnez un fichier audio contenant des chiffres en français
   - Cliquez sur "🚀 Envoyer"
   - Les chiffres apparaissent automatiquement en gellules

3. Éditez un chiffre mal détecté:
   - Cliquez sur une gellule
   - Modifiez la valeur (1-1000)
   - Appuyez sur Entrée

4. Testez le temps réel:
   - Ouvrez http://localhost:5173 dans plusieurs onglets
   - Modifiez un chiffre dans un onglet
   - Observez la synchronisation instantanée
